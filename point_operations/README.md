# Point Operations 

This module contains a report on point operations in digital image processing, developed through experimental practice using the Image Processing Laboratory software. It covers image transformations across color, grayscale, and binary formats, including arithmetic and logical operations, as well as thresholding techniques such as manual and Otsu methods.
