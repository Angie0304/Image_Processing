# Image_Processing
This repository contains projects and practical exercises in digital image processing, including point operations, mathematical morphology, and image segmentation, with a focus on implementation and analysis of techniques.



## Repository Structure

```text
image_processing/
├── mathematical_morphology/   # Binary and grayscale morphology experiments
├── point_operations/          # Arithmetic, logical operations and thresholding
├── projects/                  # Projects: Facial segmentation
└── README.md                  # RMain documentation and index
```
